Eastern volleyball remains undefeated

Originally found at https://www.bristoledition.org/?p=3021 Published on 2020-10-16 12:13:32

The Bristol Eastern volleyball team ran their record to 6-0 with a straight set victory over New Britain Thursday.

Match Results:
Bristol Eastern 3
New Britain 0

Set scores:
Set 1: 25-20
Set 2: 25-12
Set 3: 25-21

The Lancers were led by Ryley Plourde with 11 kills, Hannah Webber with 7 kills, and Emma Rodjenski with 5 kills. Alyssa Adorno added 3 aces and Hannah Podolak had 3 assists.

Bristol Eastern's next match will be at Avon on Monday, October 19.
